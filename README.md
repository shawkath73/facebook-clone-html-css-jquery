📘 Facebook Login UI Clone
A front-end Facebook login page clone built with HTML, CSS, and jQuery. This project was created as part of my web development learning journey to practice responsive design, layout structuring, and dynamic interactivity.

🔍 Features
✅ Facebook-style login form layout
✅ Styled using custom CSS
✅ Interactivity with jQuery (e.g., form validation, dynamic elements)
✅ Clean code structure for easy understanding

🛠️ Tech Stack
- HTML5
- CSS3
- JQUERY
