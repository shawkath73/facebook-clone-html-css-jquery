📘 Facebook Login UI Clone<br>
A front-end Facebook login page clone built with HTML, CSS, and jQuery.<br> This project was created as part of my web development learning journey to practice responsive design, layout structuring, and dynamic interactivity.<br>

🔍 Features<br>
✅ Facebook-style login form layout<br>
✅ Styled using custom CSS<br>
✅ Interactivity with jQuery (e.g., form validation, dynamic elements)<br>
✅ Clean code structure for easy understanding<br>

🛠️ Tech Stack<br>
- HTML5
- CSS3
- JQUERY
